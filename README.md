# 原住民就業狀況調查之抽樣設計報告

這是一份以原住民家庭戶為母體之抽樣調查設計報告，重點在於透過分層抽樣與兩階段整群抽樣，建立兼具代表性與可執行性的調查方案。

本專案為抽樣調查課程之報告，主題為「原住民就業狀況調查之抽樣設計報告」。
研究目的是針對全臺原住民家庭戶設計一套可行的抽樣方法，在有限調查成本下兼顧代表性與執行可行性。

## 專案背景
本研究以全臺原住民家庭戶為母體，並依地區特性分為三大層：
1. 山地鄉原住民
2. 平地鄉原住民
3. 非原民鄉之原住民

由於題目要求 30 個山地鄉與 25 個平地鄉皆需訪問，但非原民鄉不必全部調查，因此本研究採用：
- 分層抽樣法
- 兩階段整群抽樣法

## 研究內容
本報告主要包含以下內容：
- 抽樣設計說明
- 樣本數分配方法與配置準則
- 每戶平均收入估計公式
- 各縣市戶數與樣本配置表
- 抽樣配置的假設與計算邏輯
- 結論

## 研究方法
### 1. 分層抽樣
將母體分為三大層，分別處理不同地區型態之原住民家庭戶。

### 2. 比例配置
總樣本數設定為 11,000 戶，依各層母體戶數比例進行樣本分配。

### 3. 兩階段整群抽樣
在非原民鄉層中，先抽取鄉鎮，再於中選鄉鎮內抽取家庭戶，以降低調查成本並提升執行效率。

## 專案結果
本研究完成了一套適用於原住民就業狀況調查的抽樣設計方案，並建立全臺原住民家庭每戶平均收入之估計公式，可作為後續統計推論與實務調查執行之基礎。
